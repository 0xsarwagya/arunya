## About The Project

<!-- [![Product Name Screen Shot][product-screenshot]](https://example.com) -->

Arunya is a tool for web owners to see how users interact with their sites. Whether you want to track how many people visit, how far down a page they scroll, or which buttons they click, Arunya gives you simple, easy-to-understand analytics.

**Features**:
* **Privacy-focused**: We don't track personal data. Only aggregate, anonymous data is collected.
* **Easy Setup**: Just add a small JavaScript snippet to your site, and you’re good to go!
* **Custom Events**: You can track specific actions like button clicks, form submissions, and more.
